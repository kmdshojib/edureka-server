
# Edureka server-site

This is a server site for an educational website called Edureka.





## Tech Stack


**Server:** Node, Express, Cors

## Run is project

To install dependecies

```bash
npm install
```

To Run this project 

```bash
npm start
```


## Deployment

To deploy this project run this on cmd

```bash
  npm i -g vercel
```

```bash
  vercel
```
